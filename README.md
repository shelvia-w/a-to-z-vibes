# A to Z Vibe Coding

A small React gallery for alphabet-based creative coding projects by Shelvia Wongso.

## Development

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm start
```

Create a production build:

```bash
npm run build
```

## GitHub Pages

This repo is configured for GitHub Pages at:

```text
https://shelvia-w.github.io/a-to-z-vibes/
```

Deployment is handled by `.github/workflows/deploy.yml`. After pushing to `main`,
enable GitHub Pages in the repository settings and choose GitHub Actions as the
source.
